# Reference data — polling district sources requiring digitization

These sources contain polling district boundaries in non-GIS form and require manual
digitization (or extraction from ZIPs) to obtain polygon shapefiles for analysis.

> Large binaries (ZIPs, PDFs, PNGs) are gitignored; acquire separately.
