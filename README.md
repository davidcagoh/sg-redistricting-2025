# Singapore electoral redistricting analysis

MCMC ensemble analysis of Singapore's electoral boundary changes (2020 → 2025) using GerryChain/ReCom on URA subzones.

---

## Quick start

```bash
pip install -r requirements.txt
pip install geopandas          # needed for sanity.py only

# 1. Reproduce processed data
python3 src/merge_census_and_geospatial.py      # → data/processed/subzone_with_population.geojson
python3 src/validate_and_copy_geospatial.py     # → data/processed/electoral_boundaries_*.geojson
python3 sanity.py                               # verify all processed files

# 2. Run tests
pytest --cov=src --cov-report=term-missing      # 481 tests

# 3. Run MCMC analysis (in order)
python -m src.analysis.cli assign-actual --year 2020
python -m src.analysis.cli assign-actual --year 2025
python -m src.analysis.cli run-ensemble --run-id sg2025 [--n-steps 10000]
python -m src.analysis.cli diff \
  --run-id <diff_id> \
  --year-2020-run-id <id> \
  --year-2025-run-id <id>
```

## Data sources

| Layer | Source | Size |
|-------|--------|------|
| Population by subzone | Census 2020 (Dept of Statistics) | 3 CSVs, ~196K |
| Subzone polygons | URA Master Plan 2019 | GeoJSON, ~3MB |
| Electoral boundaries | ELD / data.gov.sg | GeoJSON, ~2MB (2020 + 2025) |
| HDB buildings | Teoalida sample | GeoJSON + CSV, ~55MB |
| Cadastral parcels | SLA | GeoJSON + KML, ~318MB |
| Community polling districts | ge2025_polling_districts | KML + JSON + ZIPs, ~609MB |

Large files (hdb, sla_cadastral, polling_community) are not tracked in git — acquire separately.

## Known gaps

- **Polling district polygons** — official boundaries not available as a GIS layer; community KML and PDFs exist in `data/raw/polling_community/` and `data/reference/pdfs_to_digitize/`. Manual digitization needed.
- **KML CRS** — raw KML files may need CRS conversion to WGS84 before use in GIS tools.
- **Census–subzone match** — some URA subzones (non-residential, small areas) have no Census row; `subzone_with_population.geojson` records `pop_total = 0` for these.
- **No partisan vote data** — precinct-level vote data is not public in Singapore; partisan analysis infeasible.

## Graph structure

332 subzone nodes, 850 rook-adjacency edges. 4 offshore islands excluded (Sudong, Semakau, Southern Group, Pulau Seletar).

Primary join key: **subzone name** (normalized to uppercase) — Census `Number` ↔ URA `SUBZONE_N`.
